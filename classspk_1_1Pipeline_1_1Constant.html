Detailed Description

Specializes UniformObject for managing global constants in Pipeline.

This class is a specialization of UniformObject designed for handling global shader constants that do not change per object or draw call.

It simplifies the process of managing constants that are common across multiple objects or the entire render pipeline.

Usage example:

spk::Pipeline::Constant& constant = pipeline.constant("MyConstant");
constant["myValue"] = 42;
constant.push();
