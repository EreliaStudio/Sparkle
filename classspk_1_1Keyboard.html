Detailed Description

Manages keyboard input states and character input for an application.

This class provides functionality to track the state of individual keys on the keyboard, including whether they are currently pressed, released, up, or down. It also handles character input, allowing for the capture of typed characters in a thread-safe manner using atomic operations.

The Keyboard class defines an enum for key codes that represent various keyboard keys, including alphanumeric keys, function keys, control keys, and more. This enables the application to respond to specific key events and manage keyboard-based interaction efficiently.

Usage example:

const spk::Keyboard& keyboard = myApplication.keyboard();
// In an input handling loop
if (keyboard.getKey(spk::Keyboard::Key::A) == spk::InputState::Pressed) {
// Respond to 'A' key press
}
wchar_t typedChar = keyboard.getChar();
if (typedChar != L'\0') {
// Handle character input
}
