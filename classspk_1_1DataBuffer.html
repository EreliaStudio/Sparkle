Detailed Description

A buffer for handling binary data with support for serialization and deserialization.

The DataBuffer class provides methods to store, modify, and retrieve binary data. It supports serialization and deserialization of standard layout types and containers. The buffer can be resized, cleared, and reset, and it maintains a bookmark for tracking read positions.
