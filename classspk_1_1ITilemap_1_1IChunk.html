Detailed Description

template<typename TNodeIndexType, size_t TSizeX, size_t TSizeY, size_t TSizeZ>
class spk::ITilemap< TNodeIndexType, TSizeX, TSizeY, TSizeZ >::IChunk

Represents a single chunk within the tilemap.

Chunks are individual segments of the tilemap, each containing a fixed number of nodes. This class provides the base functionality for a chunk, including data storage, rendering, and serialization to/from files.
