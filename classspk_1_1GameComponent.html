Detailed Description

Base class for game components that can be attached to GameObjects.

This class serves as a base class for various game components that can be attached to GameObjects. It provides a common interface for handling rendering and updating logic in a game. Derived classes should implement the onRender and onUpdate methods to define specific behavior for rendering and updating the component.

This class also inherits from ActivateObject, allowing components to control their activation state. Uppon creation, GameComponent are ste as activated.

Usage example:

class MyCustomComponent : public spk::GameComponent
{
public:
MyCustomComponent(const std::string& p_name) : spk::GameComponent(p_name) {}
void onRender() override
{
// Implement rendering logic here
}
void onUpdate() override
{
// Implement update logic here
}
};
