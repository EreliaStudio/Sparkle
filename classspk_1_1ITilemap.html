Detailed Description

template<typename TNodeIndexType, size_t TSizeX, size_t TSizeY, size_t TSizeZ>
class spk::ITilemap< TNodeIndexType, TSizeX, TSizeY, TSizeZ >

Represents a tilemap composed of multiple chunks.

This class serves as a base for tilemap implementations, utilizing a templated node index type and fixed-size chunks. It provides mechanisms to manage chunks, including creation, querying, and serialization to/from files.

Template Parameters
TNodeIndexTypeThe type of the node index used in the tilemap.
TSizeXThe width of each chunk.
TSizeYThe height of each chunk.
TSizeZThe depth of each chunk.
