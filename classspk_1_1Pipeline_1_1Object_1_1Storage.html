Detailed Description

Manages the storage for vertex and index buffers associated with a drawable object.

The Storage class encapsulates vertex and index data for an Object, handling the creation and management of vertex buffer objects (VBOs) and element buffer objects (EBOs). It supports the setup of vertex attributes based on a defined layout.

Usage example: Not directly instantiated by users, used internally by Object.
