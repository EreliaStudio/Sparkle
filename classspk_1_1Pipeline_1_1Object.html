Detailed Description

Represents a drawable object within the Pipeline, including geometry and attributes.

This class is responsible for managing the vertex array object (VAO), vertex buffer object (VBO), and any associated attributes or textures.

It encapsulates the geometry (vertices and indices) and provides methods to set attribute values and draw the object using the Pipeline that created the Object.

Usage example:

spk::Pipeline::Object object = pipeline.createObject();
object.setVertices(myVertices);
object.setAttribute("MyAttribute", value);
object.render();
