Detailed Description

A module for managing and processing collisions between game objects within the game engine.

CollisionModule extends GameEngineModule to specifically handle collision detection and response among game objects. It uses a broad-phase and narrow-phase collision detection strategy to efficiently identify potential collisions and confirm actual collisions, respectively. The module maintains a list of game objects and checks for collisions between them during the game loop's update phase.

Usage example:

spk::GameEngine engine;
spk::CollisionModule collisionModule;
engine.addModule(&collisionModule);
