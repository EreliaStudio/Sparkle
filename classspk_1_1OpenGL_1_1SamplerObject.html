Detailed Description

Manages OpenGL sampler uniforms, linking texture units to shader sampler locations.

SamplerObject abstracts the association between a shader's sampler uniform and an OpenGL texture unit, facilitating texture binding and activation within shaders. It simplifies the process of setting which texture unit a shader's sampler should use.

Usage example:

SamplerObject sampler(shader.getUniformLocation("myTextureSampler"));
sampler = 0; // Bind the sampler to texture unit 0
sampler.activate();
