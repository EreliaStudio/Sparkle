Detailed Description

template<typename TType, typename = typename std::enable_if<std::is_arithmetic<TType>::value>::type>
interface spk::IVector3< TType, typename >

Represents a 3D vector with components x, y, and z.

Template Parameters
TTypeThe numeric type used to represent the vector's components. It must be an arithmetic type (e.g., int, float, double).

This class provides a template for creating 3D vectors of various types (e.g., float, int, unsigned int). It includes common vector operations such as addition, subtraction, multiplication, division, dot product, cross product, normalization, and more.
